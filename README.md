# NTRClientJS

Exactly what it sounds like. This is a small library to interact with the NTR debugger on the 3DS in Node.

Documentation can be found [here](http://cu3po42.github.io/NTRClientJS/).
Not all commands are documented at this point, if you are interested in helping out, please send a Pull Request.
